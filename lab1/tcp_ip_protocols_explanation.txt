TCP/IP (Transmission Control Protocol/Internet Protocol) - це набір протоколів, які використовуються для забезпечення зв'язку та передачі даних в комп'ютерних мережах, включаючи Інтернет. TCP/IP розбивається на кілька основних протоколів та рівнів, кожен з яких виконує свої завдання:

Internet Protocol (IP):

IPv4 (Internet Protocol version 4): Цей протокол використовує 32-бітні адреси і є основною технологією, яка визначає IP-адреси для ідентифікації пристроїв у мережі. IPv4 має обмежену кількість доступних адрес, і це спричинило перехід до IPv6.
IPv6 (Internet Protocol version 6): IPv6 використовує 128-бітні адреси, що розширює простір IP-адрес та розв'язує проблему нестачі адрес, яка існувала у IPv4.
Transmission Control Protocol (TCP):

TCP відповідає за надійний та послідовний обмін даними між комп'ютерами у мережі. Він гарантує, що дані надсилаються і приймаються без втрат та в правильному порядку. TCP також відстежує передачу даних, відновлює втрачені пакети і контролює швидкість передачі даних. Використовується, наприклад, для передачі веб-сторінок, електронної пошти і багатьох інших додатків.
User Datagram Protocol (UDP):

UDP також використовується для передачі даних, але він є менш надійним і не гарантує послідовність чи доставку всіх даних. Використовується для додатків, які можуть пережити деяку втрату даних або не вимагають гарантованої послідовності, таких як відео- та аудіострімінг.
Internet Control Message Protocol (ICMP):

ICMP використовується для передачі повідомлень про помилки та інших керуючих повідомлень між пристроями в мережі. Найвідоміший приклад - "ping," який використовує ICMP для вимірювання часу відправки та прийому пакетів між пристроями.
Internet Group Management Protocol (IGMP):

IGMP використовується для керування та відслідковування групового (multicast) обміну даними в мережі.
TCP/IP протоколи дозволяють передавати дані в мережі, незалежно від фізичних технологій, що використовуються (наприклад, Ethernet або бездротові зв'язки). Вони становлять основу для функціонування Інтернету та дозволяють різним пристроям та системам спілкуватися та обмінюватися інформацією в різних мережах.
